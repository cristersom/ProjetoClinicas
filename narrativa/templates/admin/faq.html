1. Glossário do Sistema

Narrativa
É a entidade principal (a jornada completa). Contém o título, a descrição, a imagem de capa e agrupa todas as cenas.
Cena
Representa uma tela individual que o paciente visualiza. Pode conter texto, vídeo ou imagem. É a unidade básica de navegação.
Escolha (Botão)
Opção de navegação criada dentro de uma Cena. Permite ligar a Cena A à Cena B, criando a interatividade.
Questionário
Formulário de coleta de dados. Pode ser vinculado a uma Cena específica para interromper a narrativa e solicitar feedback do paciente.
Sessão do Paciente
Registro técnico que identifica um acesso único. O sistema usa isso para gerar relatórios de progresso e retenção.

2. Fluxo de Criação de Narrativas

O sistema utiliza um banco de dados relacional. Isso significa que você não pode vincular algo que ainda não existe. Siga esta ordem lógica para evitar erros:

Regra de Ouro: Crie o conteúdo primeiro (Cenas), depois faça as conexões (Links e Cena Inicial).
  1. Passo 1: Criar a Narrativa (Estrutura)
    Acesse "Narrativas" > "Adicionar". Preencha Título, Descrição e Capa.
    Nota: Deixe o campo "Cena Inicial" em branco neste momento. Salve.
  2. Passo 2: Criar as Cenas (Conteúdo)
    Acesse "Cenas" > "Adicionar". Crie todas as telas da sua jornada.
    Em cada cena, selecione a Narrativa criada no Passo 1. Salve cada uma.
  3. Passo 3: Criar as Conexões (Escolhas)
    Volte para editar as Cenas criadas. No final da página de edição, use a seção "Escolhas" para criar os botões.
    Agora que todas as cenas existem, você poderá selecionar a "Cena de Destino" correta no menu.
  4. Passo 4: Definir o Início
    Volte para editar a Narrativa (Passo 1). Agora selecione a "Cena Inicial" correta e salve.

3. Gestão de Questionários

Os questionários permitem coletar dados qualitativos e quantitativos durante a jornada.

Como criar e vincular:

  1. Vá em "Questionários" > "Adicionar".
  2. Defina um Título para o formulário.
  3. Cena Associada: Selecione a cena onde este questionário deve aparecer. O formulário será exibido automaticamente abaixo do conteúdo dessa cena.
  4. Cena de Destino (Opcional): Para onde o paciente vai após responder? Se deixar em branco, ele permanece na mesma tela (útil se a cena tiver botões de escolha manuais).

Tipos de Pergunta:

  • Texto Livre: Caixa de texto aberta para opiniões.
  • Escala (1 a 5): Sistema de estrelas para avaliação.
  • Única Escolha: Botões de rádio (apenas uma opção válida).
  • Múltipla Escolha: Caixas de seleção (várias opções válidas).
Dica: Para perguntas de múltipla escolha, você deve adicionar as "Opções de Resposta" clicando no botão "Adicionar outro(a) Opção" dentro da tela de edição da Pergunta.

4. Distribuição e Acesso

Menu de Narrativas (Hub)

Link que exibe todas as narrativas disponíveis. Ideal para enviar ao paciente e deixá-lo escolher.

  • No Painel Administrativo, clique no botão "Compartilhar Acesso Geral" no topo da tela.

Link Direto (Narrativa Específica)

Link que abre uma jornada específica imediatamente.

  • Clique na narrativa desejada para ver os Detalhes.
  • Use o botão cinza "Copiar Link" ou o botão verde "Encaminhar" (mobile).

5. Relatórios e Dados

Dashboard (Visão Geral)

Acesse clicando em "Dashboard" no menu lateral. Mostra KPIs de acessos totais, gráfico de retenção de 30 dias e as narrativas mais populares.

Relatório de Respostas (Exportação)

Para baixar os dados em Excel (XLSX):

  1. Vá em "Questionários".
  2. Na lista, localize o questionário desejado.
  3. Clique em "Ver Resumo".
  4. Na tela de resumo, use os botões de "Exportar" no topo para baixar a planilha consolidada (onde cada linha é um paciente).
